Die 10 günstigsten Dönerläden in Bamberg — von billig bis teuer, mit Karte.  
The 10 cheapest döner places in Bamberg — ranked, with an interactive map.  
Bamberg'deki en ucuz 10 dönerci — sıralı liste ve interaktif harita.

---

## Features

- **Rangliste / Ranking / Sıralama** — sorted cheapest to most expensive, click any address to open Google Maps
- **Karte / Map / Harita** — interactive Leaflet map, pins sized by rank (№1 = largest)
- **DE / EN / TR** language switcher, saved to localStorage
- **Dark mode** — auto-detects system preference, toggle in header
- Single `index.html` — no build step, no dependencies to install
